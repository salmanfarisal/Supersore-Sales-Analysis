# Superstore Sales Performance Analysis 📊
Proyek ini bertujuan untuk menganalisis performa penjualan dan profitabilitas sebuah toko retail (Sample Superstore) menggunakan Python.

## Masalah Bisnis
Meskipun total penjualan tinggi, perusahaan mengalami kerugian di beberapa kota besar dan kategori produk tertentu.

## Tools yang Digunakan
- **Python** (Pandas untuk pengolahan data)
- **Matplotlib & Seaborn** (Visualisasi data)
- **Google Colab** (Environment pengembangan)

## Temuan Utama (Insights)
- **Profit vs Discount:** Ditemukan korelasi negatif antara diskon dan profit. Diskon di atas 20% cenderung membuat profit menjadi negatif.
- **Kota Bermasalah:** Philadelphia dan Houston mengalami kerugian terbesar.
- **Kategori Kritis:** Office Supplies dan Furniture memiliki margin yang sangat tipis akibat kebijakan diskon yang agresif.

## Rekomendasi Bisnis
- Mengurangi intensitas diskon pada kategori Office Supplies di wilayah dengan profit negatif.
- Menerapkan batas minimum pembelian untuk penggunaan diskon besar.
